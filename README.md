\# 🧁 Cardápio Digital - Doces da Lari



🔗 \*\*Acesse o cardápio online:\*\* \[milhomesss.github.io/cardapio\_doces\_da\_lari](https://milhomesss.github.io/cardapio\_doces\_da\_lari/)



Um cardápio digital interativo, moderno e responsivo desenvolvido para a confeitaria artesanal \*\*Doces da Lari\*\*. 



Este projeto foi criado com o objetivo de facilitar a visualização dos produtos pelos clientes e agilizar o processo de encomendas, integrando o cardápio diretamente com o WhatsApp da loja.



\## 📱 Funcionalidades



\* \*\*Navegação por Abas:\*\* Interface fluida que permite ao usuário alternar facilmente entre as categorias (Bolos Confeitados, Bolos Caseiros e Doces) sem precisar recarregar a página.

\* \*\*Design Responsivo:\*\* O layout se adapta perfeitamente a diferentes tamanhos de tela (smartphones, tablets e desktops), garantindo uma boa experiência para o cliente em qualquer dispositivo.

\* \*\*Integração com WhatsApp:\*\* Um botão de "Fazer Encomenda" que redireciona o cliente direto para o WhatsApp da confeitaria, já com uma mensagem padrão preenchida para iniciar o atendimento.

\* \*\*Identidade Visual Temática:\*\* Uso de paleta de cores (tons de rosa), tipografia estilizada (Dancing Script) e elementos visuais que remetem à marca.



\## 🛠️ Tecnologias Utilizadas



O projeto foi construído utilizando tecnologias front-end puras, sem uso de frameworks:



\* \*\*HTML5:\*\* Estruturação semântica do conteúdo.

\* \*\*CSS3:\*\* Estilização avançada utilizando CSS Variables (Custom Properties), Flexbox e CSS Grid para o layout dos itens, além de transições suaves.

\* \*\*JavaScript (Vanilla):\*\* Lógica simples e eficiente para a manipulação do DOM na troca das abas e redirecionamento de links.
